# 简介

Front-End Algorithm learning series（《前端算法学习路》）主要包含对于前端学习算法各个阶段的思考。本系列的文章更加偏向于面试、工程使用，而非 ACM 竞赛向。同时也会分析更大公司的面试题，比如 Google、Microsoft、头条等。
