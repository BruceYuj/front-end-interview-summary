## 递归
正常来讲我们的递归如下：
```javascript
var sumNums = function(n) {
  if(n === 0) return 0;

  return n + sumNums(n-1);
}
```

此时递归出口使用了 if，那么还有什么可以表达 if 呢？
逻辑运算符 `&&` 或 `||`

```javascript
var sumNums = function(n) {
    let sum = 0;
    n && (sum = n + sumNums(n-1));

    return sum;
};
```
**复杂度分析**:
- 时间复杂度：$O(n)$
- 空间复杂度：$O(n)$

## 快速乘
我们可以通过位运算和加法来模拟乘法。

```javascript
var quickMulti = function(a, b) {
  let ans = 0;

  for(; b; b >>>= 1) {
    if(b&1) {
      ans += a;
    }
    a <<= 1;
  }
  return ans;
}

```

因此，我们可以得出：
```javascript
var sumNums = function(n) {
    return helper(n, n+1) >> 1;

    function helper(a, b) {
      ans = b && helper(a << 1, b >> 1);
      b&1 && (ans += a);

      return ans;
    }
};
```
**复杂度分析**:
- 时间复杂度：$O(logn)$
- 空间复杂度：$O(logn)$
